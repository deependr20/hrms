========================================
    TROUBLESHOOTING
========================================

If you can't login:
1. Make sure MongoDB is running
2. Check if database is seeded
3. Clear browser cache
4. Try different browser

If port 3001 doesn't work:
- Check terminal for actual port
- Server might be on 3000, 3002, or 3003
